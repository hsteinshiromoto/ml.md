# Bernoulli Distribution

## Definition

Let $X$ be a [[random variable]] with outcome set $\mathbb{N}$ and [[Probability Mass Function|PMF]] defined as

$$p_X:\{0,1\}\to\mathbb{R}_{\geq0}$$

$$\{0,1\}\ni x\mapsto p_X(x)=p^x(1-p)^{1-x}\;.$$
