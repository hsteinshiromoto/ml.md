# Type-I Error

## Definition

Let $\phi$ be a [[test]], the [[Statistical Test|null hypothesis]] can be rejected ($\phi(y)=1$) when it is in fact true ($\theta\in\Theta_0$). This error is said to be a _type I error_ or _false positive_ . ^ee7637
