# Compact Set

## Context

## Statements

**Definition**. ([[Citation Needed]]). Let $(X, \tau)$ be a [[Topological Space|topological space]], a set $K\subset X$ is said to be _compact_ if every [[open cover]] of $K$ has a finite [[subcover]].
