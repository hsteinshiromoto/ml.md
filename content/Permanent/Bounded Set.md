# Bounded Set

## Definition

**Definition**. Let $(X, d)$ be a [[metric space]]. A set $E\subseteq X$ is said to be _bounded_, if for every $\varepsilon>0$, there exists a finite number of [[Open Ball|open balls]] $B_{<\varepsilon}(\cdot)$ that [[Cover|covers]] $E$.
