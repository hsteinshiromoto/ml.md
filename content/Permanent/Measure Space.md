# Measure Space

A measure space is a triple ${\displaystyle (\Omega,{\mathcal {F}},P )}$ where

- $\Omega$ is a set.
- ${\mathcal {F}}$ is a [[Sigma Algebra|Ïƒ-algebra]] on the set $\Omega$.
- $P$ is a [[Measure]] on ${\displaystyle (\Omega,{\mathcal {F}})}$.
