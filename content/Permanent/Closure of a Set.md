# Closure of a Set

## Definition

**Definition**. Let $(X,\tau)$ be a [[topological space]], and $S\subseteq X$. The _closure of_ $S$, denoted as $\mathrm{cl}(S)$, is the set of all elements $x\in X$ such that every [[neighborhood]] of $x$ meets $S$.
