# Open Set

## Context

## Statements

**Definition** ([[Tu, L. W. - An introduction to Manifolds|@Tu11, pp. 318]]). Let $(X,\tau)$ be a [[topological space]], a set $S\subset\tau$ is said to be an _open set_ of $X$.
